Search


HudsonFog offers full-text search and property-based search.

Full-text Search

This is a traditional keyword-based search common on the Web. We index HTML, text documents as well as Office documents like Excel spreadsheets. We use a powerful and very flexible search engine Lucene.

Property-based Search

Property-based search is extremely powerful. It allows to pin-point the requested resources based on the combination of the desired properties. In addition to searching HudsonFog site property-based search allows to search Business Networks powered by HudsonFog.
