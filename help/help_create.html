Add Items - creating new items in semory
 
  Layout Every item add page is laid out the same way. The item properties simply need filling in, following any instructions beside each box.
*
 Required These must be filled in to save an item. Without them the item is incomplete.
List options Select an item by clicking on the downward arrow then click the item required. Some list options are too large to display so you will see an option at the bottom of the list saying "more..". Click this and you will see an item list showing all the options.
Dates Date fields will require a correctly entered date in the form dd-mm-yyyy. You can also click on the calendar link to select a date and let it fill the box for you.
  Saving Once you have entered all the details click on the submit button to save the item. Or click cancel and button to return to the list of items.
Help Where applicable each property has a help icon beside the name. Click on the icon to read the description of what's required in the box.
