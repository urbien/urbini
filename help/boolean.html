Booleans - ask for more than one thing
     
All words red AND blue Use AND to find resources that contain all the words you're searching for. This example will only find things with both red and blue in.
Any words red OR blue Entering multpile words seperated by OR will find results with any of the words you entered. This will find anything with either red or blue. OR is the default search and isn't normally required.
Not this word NOT red Adding NOT before a word will prevent results if they contain the word. This example will only show results that don't contain the word red.
Must include +red blue This allows you to find things that must include a certain word. This example finds things that contain the word red and can include blue.
Must not include +red -blue This allows you to find things that must not include a certain word. This example finds things that might contain the word red but does not include the word blue.
